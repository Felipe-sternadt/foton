# 🚛 Foton Sales Page Premium

Landing page moderna e interativa desenvolvida para apresentação de caminhões Foton, com foco em experiência premium, animações suaves e design sofisticado.

---

## ✨ Features

- 🎬 Scroll cinematográfico com travamento de seções
- 🚀 Animações suaves baseadas em scroll
- 🧠 UX pensada para sensação premium
- 📱 Layout responsivo
- 🎯 Estrutura otimizada para conversão

---

## 🛠️ Tecnologias

- HTML5
- CSS3 (Custom Properties + Animations)
- JavaScript (Vanilla)
- Intersection Observer API
